# 🔥 VolixChat Server

السيرفر الخلفي لتطبيق **VolixChat** - تطبيق تواصل اجتماعي آمن وسريع.

## 🌐 الرابط الرسمي
[volixchat.com](https://volixchat.com)

## 🛡️ المميزات
- تشفير من طرف لطرف (E2E)
- WebSocket للمحادثات الفورية
- JWT للمصادقة
- حماية SSL/TLS
- Node.js + Express

## 🚀 التشغيل

```bash
npm install
npm start
